# Beautiful Malino

Website reservasi objek wisata di Malino

## Description

Website ini adalah website untuk melakukan reservasi paket wisata yang ada di Malino.

## Tech Used
* PHP
* JavaScript
* Bootstrap

## Features
* Lihat reservasi
* Buat reservasi
* Update reservasi
* Hapus reservasi
